Project Spotlight:

This month's project spotlight features my e-commerce website—the first version of many to come! This platform allowed me to advance my proficiency in HTML, CSS, JavaScript, and JSON file integration. As a collector myself, I understand the passion behind the trend. This initial release is just the beginning—I'm excited to expand its features as my technical skills continue to grow.
